# Hoop Activity Planner

Hoop Activity Planner is an interactive webpage created for a DOM manipulation assignment. The project helps users plan basketball training activities by adding activities to a list, removing them, searching activity cards, and switching between light and dark mode.

## Features

- Navigation bar with dropdown menu
- Hamburger menu for smaller screens
- Change message button
- Dark and light mode toggle
- Add custom activity using an input field
- Remove activity by clicking it
- Activity cards with buttons
- Search/filter activities
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation
- Git and GitHub

## DOM Manipulation Used

This project uses JavaScript DOM methods such as:

- `getElementById()`
- `querySelectorAll()`
- `addEventListener()`
- `classList.toggle()`
- `createElement()`
- `appendChild()`
- `remove()`
- `textContent`
- `style.display`
