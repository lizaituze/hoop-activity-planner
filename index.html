    About This Project

    Hoop Activity Planner is an interactive basketball training webpage created for a DOM manipulation assignment. The project helps users plan their basketball activities, add custom activities, remove activities, search through training cards, and switch between light and dark mode.

    Project Purpose

    The purpose of this project is to show how JavaScript can use the DOM to make a webpage interactive.

    Basketball Focus

    The page is based on basketball training activities such as shooting, fitness, ball handling, and teamwork.

    DOM Features

    The project uses buttons, input fields, search filters, list creation, dark mode, and menu interactions.

    How To Use This Page

    1. Add Your Own Activity

    Type a basketball activity in the input field and click Add Activity. It will appear in your training list.

    2. Choose From Cards

    You can also click the Add Activity button on any training card to add that activity to your list.

    3. Remove Activities

    Click any activity in your training list to remove it from the page.

    4. Search Activities

    Use the search box to quickly filter the activity cards and find the training activity you want.
